# Restaurant_Management
This is a e-commerece project for restaurants on which customers can order and admin can track all the data on froent side.
Technologies used- HTML , CSS , JS for frontend and  PHP and Mysql for backend.
Use phpMyadmin database for this and Xampp server for the localhost.
there are two portals (customer and admin )
Customer can signup and login after that order food from the restaurant.
Admin can track the order details and edit menu on the side and have all the access to change menu, proice , product details etc.
